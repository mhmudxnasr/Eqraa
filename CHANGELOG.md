# Changelog

All notable changes to this project will be documented in this file.

## [2.1.0] - 2026-01-11
### Added
- **Supabase Cloud Sync**: Real-time synchronization of reading progress, highlights, and user preferences.
- **Unified Reader Settings**: A modern, cohesive settings panel for Typography, TTS, Search, and Sync, inspired by Google Stitch design.
- **Full Backup**: Manual and automated full backup of library data to Supabase Storage.
- **Improved UX**: New Material 3 UI components, sync feedback via SnackBar, and consolidated reader controls.
- **Bug Fixes**: Resolved various compilation and resource issues for a more stable build.
