# Contributing to Eqraa Reader

Thank you for your interest in contributing to **Eqraa Reader**! We aim to build the best reading experience for Arabic literature with modern AI assistance.

We welcome all types of contributions, including:
- Improving English and Arabic documentation.
- Submitting bug reports and feature requests.
- Refining AI prompts and translation accuracy.
- Contributing code for new reading features or UI enhancements.

## Getting Started

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Submit a Pull Request with a clear description of your changes.

## Development Standards

### Code Style
We use [`ktlint`](https://github.com/pinterest/ktlint) to ensure consistent Kotlin formatting. Before submitting code, please run `make format` from the project's root directory.

### AI Companion
If you are modifying the AI functionality, ensure that prompts remain concise and maintain the "Real World Vibe" analogy format.

### Eqraa Foundation
This project is built on the Eqraa Eqraa Toolkit. If you are making deep changes to the rendering engine, please refer to the [Eqraa documentation](https://readium.org).
