# Eqraa Reader

**Eqraa Reader** is an advanced Arabic EPUB reader featuring an AI companion, real-time cloud synchronization, and a modern Material 3 user interface. 

## Features

- **Cloud & Real-time Sync**: Powered by Supabase, your reading progress, highlights, and settings are synced instantly across all your devices.
- **AI Reading Companion**: Integrated with Groq AI to provide concise text analysis, plot context, and real-world analogies for everything you read.
- **Unified Settings UI**: A stunning, modern settings panel designed with Google Stitch and built with Jetpack Compose.
- **Advanced Typography**: Deep control over Arabic-optimized fonts like Literata and Vazirmatn.
- **Full Library Backup**: Secure your data with easy cloud backups.
- **Reading Themes**: Paper, Sepia, and Dark modes for the ultimate reading comfort.
- **Arabic Dictionary**: Specialized context-aware dictionary overlay.
